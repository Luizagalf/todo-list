# Description

To do list with localStorage.

- Apative layout
- Form with validation
- Adding, saving, deleting and marking tasks in localStorage

# Technologies:

- Vue
- Typescript
- Vuex
- Lodash
- Sass
- Vuelidate
- Vue-uuid
- Eslint, prettier

# To start the project, you need to run next commands:

- `npm i`
- `npm run serve`
